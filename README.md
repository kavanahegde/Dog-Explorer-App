# 🐶 Dog Explorer App

## 📌 Project Overview
Dog Explorer App is a simple web application that uses a public Dog API to display dog breeds and random dog images.
The app allows users to explore different breeds and view adorable dog pictures.

## 🌐 Live Features

### 🐕 Dog Breeds Page
- Fetches data from a public API
- Displays a list of dog breeds

### 📸 Dog Images Page
- Fetches random dog images
- Includes a "Load More" button

### 🏠 Navigation
- Users can switch between Home, Breeds, and Images pages

## 🔗 API Used
Dog CEO API:
- https://dog.ceo/api/breeds/list/all  
- https://dog.ceo/api/breeds/image/random/8  

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript (Fetch API, Async/Await)
- GitHub

## 🚀 How to Run
1. Open repository:
   https://github.com/kavanahegde/dog-explorer-app  
2. Open `index.html` in browser  
3. Navigate pages
